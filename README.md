# gym-mangenent-system

A Gym Management System is a web-based application designed to simplify and automate daily gym operations such as member registration, membership plans, attendance tracking, trainer scheduling, payments, and reporting. This system helps gym owners and staff manage data efficiently, reduce manual work, and provide a smoother experience for members. It also ensures accurate record-keeping, faster check-ins, organized class scheduling, and improved overall management.

## Project overview

**Name:** gym-management-system

**Short description:** A full-stack web application to manage gym members, memberships, attendance, trainers, classes, payments and reporting. Suitable as a learning project or a foundation for a production app.

**Suggested tech stack (flexible):**

* Frontend: React + Tailwind CSS (or plain HTML/CSS/Vanilla JS for simpler builds)
* Backend: Node.js + Express (or Python + Flask/Django)
* Database: PostgreSQL (or SQLite for development)
* Auth: JWT (or session-based)
* Dev tooling: Vite / Create React App, ESLint, Prettier, Husky

**Core features:**

* User roles: Admin, Staff, Trainer, Member
* Member management: add/edit/delete, profile photos
* Membership plans: create/manage plans, set durations, prices
* Attendance check-in (QR code or manual)
* Class/training schedule with bookings
* Trainer management
* Payments and invoices (basic integration)
* Reporting: monthly revenue, active members, attendance stats
